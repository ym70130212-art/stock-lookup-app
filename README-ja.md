# 株価一覧アプリ（日本株・GPT貼り付け向け）

## できること
- 1行に1つ、銘柄コードまたは企業名・略称を入力
- 日本株の銘柄コード・企業名・株価・前日比を一覧表示
- 次の形式でそのままコピー可能

```text
9432 日本電信電話(NTT) 154.2円 前日比+1.8円(+1.18%)
```

## 注意
- 株価データは Yahoo Finance 系の非公式データに依存します。
- 遅延・取得失敗・仕様変更の可能性があります。
- 重要な判断前には証券会社画面で確認してください。

## 開発手順
1. Node.js をインストール
2. このフォルダを開く
3. `npm install`
4. `npm run dev`
5. ブラウザで `http://localhost:3000` を開く

## 銘柄を追加する方法
`data/jp-stocks.json` に1行追加してください。

例:
```json
{ "code": "8058", "name": "三菱商事", "aliases": ["三菱商事", "mitsubishi corp"] }
```
